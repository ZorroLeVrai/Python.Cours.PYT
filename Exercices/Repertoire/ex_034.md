## Déplacement de fichiers en Python

**Objectif** : Dans cet exercice, vous allez écrire un script Python qui déplace tous les fichiers commençant par `test_` et ayant l'extension `.py` depuis le répertoire `Repertoire` vers un sous-répertoire nommé `Repertoire/tests`.

1. Si le sous-répertoire `tests` n'existe pas, vous devrez le créer avant d'y déplacer les fichiers.

2. Vous disposez d'un répertoire appelé `Repertoire` qui contient plusieurs fichiers, dont certains commencent par `test_` et ont l'extension `.py`. Le but est de déplacer uniquement ces fichiers vers un sous-répertoire nommé `tests` qui se trouve dans `Repertoire`.

3. Utilisez la fonction `os.rename()` pour déplacer les fichiers.

4. N'oubliez pas d'utiliser les modules appropriés de la bibliothèque standard de Python.
